# Netwrok-analysis---psychometric-analysis
This repository aims to apply various network analysis approaches to psychometric data in order to uncover meaningful patterns and insights

A Gaussian Graphical Model (GGM) is a network model that represents the conditional dependencies among a set of continuous variables assumed to follow a multivariate Gaussian distribution. In a GGM, nodes represent variables and edges represent non-zero partial correlations between pairs of variables after controlling for all other variables in the network.

Be cautious: 
Since $\gamma$ strongly influences whether edges are included in or excluded from the graph, choose it wisely; otherwise, ending up with an empty graph is commonplace.
